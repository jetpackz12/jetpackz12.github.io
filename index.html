JDMS ( Jetz Dometory Management System )

JDMS is a web system designed to streamline and simplify the management of dormitories, hostels, or student accommodations. It is particularly useful for educational institutions, universities, colleges, and housing facilities that provide lodging for students, staff, or guests. This software application improves efficiency, reduces administrative workload, enhances security, and provides residents with a seamless experience when it comes to accommodation and related services. It is a valuable tool for educational institutions and housing facilities looking to optimize their dormitory operations. The JDMS was developed using technologies such as HTML, CSS, JavaScript, Boostrap, jQuery, Ajax, PHP, MySQL

Chea's Cake

Chea's Cake is a web system that simplifies the process of ordering and managing cake orders from customers. This system streamlines the entire process, from order placement to fulfillment, and enhances customer satisfaction by providing a convenient and efficient way to order cakes. It's a valuable tool for bakeries and confectioneries looking to expand their online presence and manage orders more effectively. This was developed using technologies such as HTML, Vanilla CSS, Vanilla JavaScript (No Backend at the moment, Only Frontend)
